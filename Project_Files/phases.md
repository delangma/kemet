\### Objectif

Fournir les outils invisibles/semi-privés qui ne peuvent pas se gérer à vue sur le plateau physique, en multijoueur en ligne avec Firebase.

\- React + Vite

\- Firebase Realtime Database

\- Tailwind CSS v3
