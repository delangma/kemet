# Règles de l'IA V2 — Kemet (Liste exhaustive des règles candidates)

## Format
Chaque règle possède 3 colonnes :
  - **Règle** : condition d'application
  - **Décision** : action ou choix favorisé si la règle est valide
  - **Poids** : valeur ajoutée au poids de base (1) du candidat correspondant

Ce fichier est une liste exhaustive de TOUTES les règles envisageables pour chaque décision.
Il sert de référence pour remplir Rules_IA.txt avec les règles retenues.
Les poids indiqués sont des suggestions à calibrer.

## Notes d'implémentation

- Ce fichier V2 liste toutes les règles ENVISAGEABLES — toutes ne sont pas forcément à implémenter.
- Les poids sont des suggestions à calibrer par test.
- Poids de base : 1 par candidat (avant application des règles).
- Les poids de règles s'additionnent si plusieurs règles s'appliquent au même candidat.
- Pour les sections "idem" (ex: Téléportation Carte ID = Téléportation Action), utiliser la même implémentation.
- Certaines règles nécessitent de connaître l'état des adversaires (VP, ank, unités) : bien passer ces données à chaque fonction de décision.
